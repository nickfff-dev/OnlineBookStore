About

Bookstore is a web application that allows users to view and purchase books. It is built using Flask, a Python web framework, and Tailwind CSS, a utility-first CSS framework. The application is built using the Model-View-Controller (MVC) architecture. The application has the following features:

    Features

  • Users can view books by category
  • Users can view book details
  • Users can register and login
  • Users can add books to their cart
  • Users can view their cart
  • Users can checkout
