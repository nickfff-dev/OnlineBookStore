The Online Book Store

I present to you the Online Book Store from where users can register and browse through a rich catalog of books which they can purchase and have them delivered to their doorsteps.

Get started Explore

The application has the following features:

  • Authentication and registration

    Users can register and login

    Users can view their profile

  • Explore

    Users can view books by category

    Users can view book details

  • Users can shop

    Users can add books to their cart

    Users can view their cart

    Users can checkout

    Users can view their order history

About

OnlineBookStore is a web application that allows users to view and purchase books. It is built using Flask, a Python web framework, and Tailwind CSS, a utility-first CSS framework. The application was solely built by Nickson as fullfilment of the alx software engineering course project The application is built using the Model-View-Controller (MVC) architecture
